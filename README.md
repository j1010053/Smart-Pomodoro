# Smart Pomodoro 手機長期試用版 v1

Smart Pomodoro 是一個幫助你「先開始，再慢慢整理」的個人工作助理。長期試用版會部署在公開 HTTPS 網址，手機不必與電腦連線，也不需要讓電腦保持開機。

## 使用方式

1. 使用手機開啟試用網址。
2. 首次開啟可按「載入範例任務」，用兩分鐘了解主要功能。
3. 從瀏覽器選單將 App 加到主畫面。
4. 之後直接從手機主畫面開啟使用。

### 加到主畫面

- iPhone：使用 Safari 開啟，按「分享」→「加入主畫面」。
- Android：使用 Chrome 開啟，按選單 →「安裝應用程式」或「加到主畫面」。

## 試用版功能

- 快速收件匣：輸入一句話就能記下任務。
- 下一步建議：依截止時間、重要性與既有進度提供理由清楚的建議。
- 五分鐘啟動：降低開始困難任務時的阻力。
- 獨立專注頁：支援暫停、繼續、提前結束、跳過、短休與長休。
- 任務管理：可編輯分類、重要性、能量、截止日與預估時間，完成後也能復原。
- 今日容量：保留緩衝，避免把一天排得太滿。
- 七日工作量：顯示截止壓力、每日需求與跳過後增加的工作量。
- 四象限負荷：分開檢視重要與緊急工作的今日占比。
- 七日回顧：顯示專注分鐘、完成率、番茄數與三項工作組成警示。
- 離線開啟：成功載入一次後，主要介面與本機資料可在離線時使用。
- 備份與還原：可匯出或匯入 JSON 備份。

## 資料保存與隱私

- 任務、計時紀錄與設定會保存在手機瀏覽器的本機資料庫。
- 手機與電腦不需要連線，也不會自動同步資料。
- 此版本不需要帳號，不會將任務上傳至雲端，也不使用 AI。
- 清除瀏覽器網站資料、移除瀏覽器或更換手機，可能造成資料遺失。
- 建議固定從設定頁匯出 JSON 備份，並保存到手機檔案或個人雲端硬碟。

## 建議的首次試用流程

1. 載入三個範例任務。
2. 查看「下一步建議」與推薦理由。
3. 開始一次五分鐘啟動。
4. 記下一件自己的任務。
5. 調整今天預計投入的時間，查看容量與四象限變化。
6. 從設定匯出一次備份，確認長期資料保護流程。

## 長期試用時請留意

- 每週至少備份一次；重要資料變更後可立即備份。
- iPhone 請避免使用無痕瀏覽，並以 Safari 加入主畫面。
- Android 請避免使用無痕模式，並以 Chrome 安裝 App。
- 若重新部署新版本，正常情況下本機資料會保留；仍建議升級前先匯出備份。

## 回饋重點

1. 能否在 10 秒內記下一件任務？
2. 下一步建議是否合理，理由是否容易理解？
3. 五分鐘啟動是否降低開始工作的阻力？
4. 容量與四象限是否幫助你減少過度安排？
5. 使用一週後，是否願意繼續從手機主畫面開啟？
6. 哪個操作最容易卡住或找不到？

## 開發與驗證

```powershell
npm run dev
npm run build
npm run test
```

正式建置結果會輸出至 `dist/`。手機長期試用應使用已部署的 HTTPS 網址，不使用區網開發網址。
